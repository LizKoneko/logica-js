# Jogo JS: Adivinhe o número!

Esse projeto é um jogo de adivinhação feito com JS, onde o objetivo é acertar qual o número que foi gerado aleatoriamente, entre 1 e 1000.

## Ferramentas usadas

* HTML

* CSS

* JavaScript
